# Old Books Buy, Sell, and Rent Platform

## Description
A peer-to-peer marketplace for buying, selling, and renting second-hand books.

## Tech Stack
- Frontend: React
- Backend: Node.js / Express
- Database: PostgreSQL / MySQL

## Modules
- Authentication
- Book Listing Management
- Buy & Rent Management
- Order Tracking
- Admin Panel

## Architecture
Controller → Service → Repository → Database
